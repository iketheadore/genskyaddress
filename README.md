# Tool for generating skycoin addresses

Randomly generate seeds and addresses.

```sh
$ genskyaddress -h

Usage of ./genskyaddress:
  -e int
        bip39 entropy, can be 128, 256 (default 128)
  -n int
        number of address need to generate (default 1)
```
